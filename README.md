---- Quick guide for Recommendation template. ----

1. Go to overleaf.com and upload the entire "dataFromStudentForLOR.zip" file 
2. Open main.tex and compile to view the contents.
3. Modify ONLY the 3 .tex files in the "studentTexFiles" folder to include your information.
    - A boilerplate main.tex file is provided that will show you how the recommendation is supposed to look like.
    - In the main.tex file, no need to modify the first paragraph, last paragraph or make the signature for your recommender. These are available to your recommender.
4. Make sure to include \ after the use of any newcommand from the basicInfo.tex file when a space is required after the information used. Check the boilerplate information in the respective tex files for information.
5. Make sure to recompile and check the draft content for spelling, grammar and formatting accuracy.
